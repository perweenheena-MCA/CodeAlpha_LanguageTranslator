# 🌍 Language Translator

This is a simple Language Translator built using Python and Tkinter.

## 🚀 Features
- Translate text into multiple languages
- Simple GUI interface
- Error handling

## 🛠️ Technologies Used
- Python
- Tkinter
- deep-translator

## ▶️ How to Run
1. Install dependencies:
   pip install deep-translator

2. Run:
   python translator_gui.py
