# 🌍 Language Translator

## 📌 Description
This is a simple Language Translation Tool built using Python and Tkinter GUI.

## 🚀 Features
- Translate text into multiple languages
- User-friendly GUI interface
- Error handling

## 🛠️ Technologies Used
- Python
- Tkinter
- deep-translator

## ▶️ How to Run
1. Install dependencies:
   pip install deep-translator
2. Run:
   python translator_gui.py
